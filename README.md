# ML Algorithms Practice Repository

This repository contains notebook-based practice implementations of common machine learning algorithms. Each notebook focuses on a single algorithm, with explanations, example datasets, model training, evaluation, and visualization where applicable.

## Repository Structure

- `01_linear_regression.ipynb` - Linear Regression
- `02_logistic_regression.ipynb` - Logistic Regression
- `03_knn.ipynb` - k-Nearest Neighbors (k-NN)
- `04_decision_tree.ipynb` - Decision Tree
- `05_random_forest.ipynb` - Random Forest
- `06_svm.ipynb` - Support Vector Machine (SVM)
- `07_naive_bayes.ipynb` - Naive Bayes
- `08_xgboost.ipynb` - XGBoost
- `09_lightgbm.ipynb` - LightGBM
- `10_kmeans.ipynb` - K-Means Clustering
- `11_pca.ipynb` - Principal Component Analysis (PCA)

## How to Use

1. Open the repository in Jupyter Notebook, JupyterLab, or VS Code with notebook support.
2. Open a notebook for the algorithm you want to study.
3. Run the notebook cells to follow the implementation and analysis.

## Goals

- Learn practical implementations of classical and modern ML algorithms.
- Compare model behavior and performance across different methods.
- Build a personal reference of notebook-based ML practice.

## Notes

- The notebooks are intended for learning and experimentation, not as production-ready pipelines.
- Datasets used in the notebooks may be built-in examples or synthetic samples generated inside each notebook.
